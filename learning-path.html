🚀 Öyrənmə Yolu PRO

A1-dən C2 səviyyəsinə qədər Alielenglish tərəfindən hazırlanmış qızıl qaydalar və xüsusi mövzularla sürətli inkişaf edin.

✨ 8 Qızıl Qayda

🧠

1. Vocabulary — Söz Ehtiyatı

Sözləri təkcə əzbərləməyin, onları kontekstdə öyrənin. Gündəlik xəbərlər oxuyun, lüğət proqramlarından istifadə edin.

📚

2. Grammar — Qrammatika

Qrammatik qaydaları əzbərləməkdən çox, praktika ilə istifadəsini öyrənin və cümlə qurun.

🎧

3. Listening — Dinləmə

Hər gün 15–20 dəqiqə ingilis dilində podcast və ya TED Talks dinləyərək dinləmə bacarığınızı inkişaf etdirin.

🗣️

4. Speaking — Danışıq

Özünüzlə danışın, səsinizi yazın və AI ilə dialoqlara girərək özünüzü sərbəst hiss edənə qədər məşq edin.

📖

5. Reading — Oxuma

Maraq dairənizə uyğun ingiliscə məqalələr, qısa hekayələr və kitablar oxumağa başlayın.

✍️

6. Writing — Yazma

Hər gün öyrəndiyiniz yeni sözlərlə qısa cümlələr və ya gündəlik yazaraq yazmaq bacarığınızı gücləndirin.

⏱️

7. Consistency — Davamlılıq

Həftədə bir gün saatlarla oxumaq əvəzinə, hər gün 30 dəqiqə ayırmaq daha effektivdir.

🎯

8. Goal Setting — Məqsəd Qoyma

Təhsil planınızı hazırlayın (məsələn, IELTS, iş üçün) və reallığa əsaslanan hədəflər qoyun.

📋 Səviyyələr üzrə Kurikulum

Beginner — Başlanğıc
A1

📚 Qrammatika

  • To be (am, is, are)
  • Present Simple Tense
  • Cəm şəkilçiləri (-s, -es)
  • This / That / These / Those
  • There is / There are
  • Artikllər: a, an, the

🗣️ Danışıq Mövzuları

  • Özünü təqdim etmə
  • Ailə üzvləri və peşələr
  • Gündəlik rutin
  • Rəqəmlər, vaxt, günlər
  • Qida və içkilər
  • Rənglər və əşyalar
Elementary — Əsas
A2

📚 Qrammatika

  • Past Simple Tense
  • Future: will / going to
  • Müqayisə dərəcələri (comparative, superlative)
  • Present Continuous Tense
  • Sayılan / Sayılmayan isimlər
  • Some / Any / Much / Many

🗣️ Danışıq Mövzuları

  • Keçmiş tətil təcrübələri
  • Gələcək planlar və xəyallar
  • Hobbilər və maraqlar
  • Alış-veriş və qiymətlər
  • Hava haqqında danışma
  • Yol göstərmə və istiqamətlər
🔒

Intermediate — Orta PRO
B1

📚 Qrammatika

  • Present Perfect Tense
  • Conditionals (Type 0, 1, 2)
  • Modal Verbs (can, could, must, should)
  • Passive Voice (əsas formalar)
  • Relative Clauses (who, which, that)
  • Gerund vs Infinitive

  • Texnologiya və internet
  • İş mühiti və müsahibələr
  • Sağlamlıq və idman
  • Fikir bildirmə və müzakirə
  • Problemlərin həlli

📚 Qrammatika

  • Reported Speech (vasitəli nitq)
  • Conditionals (Type 3, Mixed)
  • Advanced Passive Constructions
  • Wish / If only
  • Phrasal Verbs (mürəkkəb fellər)

🗣️ Danışıq Mövzuları

  • Mücərrəd mövzularda müzakirə
  • Peşəkar prezentasiyalar
  • Mədəniyyətlərarası fərqlər
  • İdiomlər və söz birləşmələri
  • Media savadlılığı
  • Ətraf mühit və sosial məsələlər
